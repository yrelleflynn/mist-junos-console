CHECK: Mist Agent Version
ID: mist-agent
File: src/checks/mist-agent-version.check.ts

PURPOSE
Checks that the Mist Agent (JMA) is installed and reports its version.
A missing or outdated Mist agent is a direct cause of cloud connectivity
failures. This check verifies the JMA package is present in the JUNOS
software.

COMMANDS
  show version | match mist

PASS
  Output contains "Mist Agent [vX.X.X]" or similar. Version string
  is reported.

FAIL
  Output is empty or does not contain "mist" — the Mist Agent package
  is not installed on this switch.

WARN
  Output mentions mist but a version number could not be parsed (the
  raw output is reported instead).

REMEDIATION
  - If not installed: install the Mist agent package. Download from the
    Mist dashboard (Organization > Inventory > Switch) or via ZTPE.
  - Run: request system software add <jma-package.tgz>
  - After install, verify with: show version | match mist
  - If version is outdated: update via Mist dashboard firmware upgrade.
